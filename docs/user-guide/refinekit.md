# RefineKit

RefineKit is the interactive trajectory proofreading tool, launched via `refinekit`.

## Purpose

Review and correct tracked trajectories with an interactive canvas, synchronized video views, and a suspicion-based review queue.

## Launch

```bash
refinekit
```

## Workflow

1. Load tracked trajectories and source video.
2. Review flagged suspicious segments in the suspicion queue.
3. Correct identity assignments using the interactive canvas and timeline editor.
4. Export refined trajectories.

## Key Features

- Suspicion queue for prioritized review of likely errors
- Synchronized multi-video grid for cross-referencing
- Interactive canvas for identity correction
- Timeline editor for temporal navigation and segment review
